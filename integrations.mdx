## WordPress Gallery Integration {#wordpress}

PostGlider can power AI-powered image search on your WordPress website. Visitors search your image gallery using natural language — "show me red dragons", "large back pieces", "US Navy tattoos" — and get instant results drawn from your PostGlider Media Vault.

### How It Works

Every image in your vault is automatically tagged by AI with visual subjects, styles, colors, and service categories. The **PostGlider Adapter** WordPress plugin connects to those tags and makes them searchable via SearchIQ.

```mermaid
flowchart LR
    A[Visitor searches "outdoor patio"] --> B[WordPress + SearchIQ]
    B --> C[PostGlider Adapter plugin]
    C --> D[gallery-search edge function]
    D --> E[user_images table — tags + search_tsv]
    E --> F[Matching images returned]
```

### Getting Your Token

Every PostGlider account has a gallery token generated automatically at signup.

<Steps>
  <Step title="Go to Settings → WordPress Gallery Integration" icon="settings" title-type="p">
    Your active token is shown in masked form with its creation date and last-used date.
  </Step>
  <Step title="Click Generate Token" icon="key" title-type="p">
    The full token is displayed **once only**. Copy it immediately.

    <Callout kind="alert">
      The token is only shown in full at this moment. Once you navigate away, only the masked version is visible. Store it in your password manager or `.env` file immediately.
    </Callout>
  </Step>
</Steps>

### Installing the Plugin

<Steps>
  <Step title="Download pg-install.php" icon="download" title-type="p">
    Download the self-installing script from the [pg-media-vault GitHub release](https://github.com/kenlyle2/pg-media-vault).
  </Step>
  <Step title="Upload to WordPress root" icon="upload" title-type="p">
    Place `pg-install.php` in the same folder as `wp-config.php` using your host's File Manager.

    <Callout kind="info">
      This method is required for Hostinger sites — Hostinger's WordPress admin blocks standard zip uploads. The self-installer bypasses this.
    </Callout>
  </Step>
  <Step title="Run the installer" icon="play" title-type="p">
    Visit `https://yoursite.com/pg-install.php` in your browser. The script creates the plugin files and deletes itself.
  </Step>
  <Step title="Network Activate" icon="check-circle" title-type="p">
    Go to **Network Admin → Plugins → Network Activate** the PostGlider Gallery Adapter. All future updates happen through the standard WordPress update mechanism.
  </Step>
  <Step title="Enter your credentials" icon="key" title-type="p">
    Go to **WP Admin → Settings → PostGlider** on your subsite. Enter:
    - Your Supabase project URL
    - Your gallery token (copied from PostGlider Settings)
  </Step>
  <Step title="Configure SearchIQ" icon="search" title-type="p">
    In SearchIQ settings, set the data source to:
    `https://yoursite.mypostglider.website/wp-json/postglider/v1/search`

    Set the query parameter to `q`. Map fields: title, description, image, url.
  </Step>
</Steps>

### Custom Subdomain Mapping

You can give a client their own branded gallery URL — like `portfolio.clientname.com` — while the site still runs on PostGlider's infrastructure.

Your client adds one DNS record (a CNAME) at their domain registrar pointing the subdomain to your network address (`network.mypostglider.site`). WordPress Multisite handles the rest natively — no mapping plugins needed.

## UploadPost — Social Publishing

PostGlider connects to UploadPost for direct social publishing. Once a Week Pack reaches `ready_to_export`, authorize UploadPost with a single click and post to all connected platforms.

Supported platforms: Instagram, Facebook, TikTok, YouTube, X (Twitter), LinkedIn.

Posts can be scheduled for a specific date and time or published immediately.

## Google Business Profile

Connect your GMB account from the **Media Vault** page to import your Google Business photos directly.

<Steps>
  <Step title="Click Import from Google in the vault header" icon="map-pin" title-type="p">
    A one-time OAuth authorization opens in a popup.
  </Step>
  <Step title="Authorize PostGlider" icon="check-circle" title-type="p">
    Grant read access to your GMB photos. PostGlider never writes to your Google account.
  </Step>
  <Step title="Photos import automatically" icon="download" title-type="p">
    All photos are imported, quality-scored, and tagged. They appear in your vault under **My Uploads** within a few seconds.
  </Step>
</Steps>

You can re-import at any time to pull in new photos added to your GMB listing.
