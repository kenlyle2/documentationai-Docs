## Webhook Configuration

Set up webhooks to receive events like `post_generated` or `post_published`.

### Steps to Configure

<Steps>
  <Step title="Generate Webhook URL" icon="link">
    In dashboard: `https://dashboard.postglider.com/webhooks/new`.
  </Step>
  <Step title="Subscribe to Events" icon="bell">
    Select events and paste your endpoint URL.
  </Step>
  <Step title="Verify Signature" icon="shield">
    Use HMAC SHA-256 with your webhook secret.
  </Step>
</Steps>

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const crypto = require('crypto');
  app.post('/webhook', (req, res) => {
    const signature = req.headers['x-postglider-signature'];
    const hash = crypto.createHmac('sha256', 'YOUR_WEBHOOK_SECRET')
      .update(JSON.stringify(req.body))
      .digest('hex');
    if (signature === `sha256=${hash}`) {
      console.log('Post generated:', req.body);
      res.status(200).send('OK');
    }
  });
  ```
  ```python
  import hmac
  import hashlib
  from flask import Flask, request

  app = Flask(__name__)
  SECRET = 'YOUR_WEBHOOK_SECRET'

  @app.route('/webhook', methods=['POST'])
  def webhook():
      signature = request.headers.get('X-Postglider-Signature')
      body = request.get_data()
      expected = hmac.new(
          SECRET.encode(), body, hashlib.sha256
      ).hexdigest()
      if signature == f'sha256={expected}':
          print('Post generated:', request.json)
          return 'OK', 200
  ```
</CodeGroup>

<ParamField query="event" param-type="string" required="true">
  Event types: `post_generated`, `post_published`, `error`.
</ParamField>

## Troubleshooting

<ExpandableGroup>
  <Expandable title="Common Integration Errors" default-open="true">
    - **401 Unauthorized**: Regenerate API tokens and ensure scopes match.
    - **Webhook 4xx**: Check endpoint accessibility from `https://api.postglider.com`.
    - **Rate Limits**: Space out posts; view limits in dashboard.
  </Expandable>

  <Expandable title="Test Integration">
    Use PostGlider's test mode: Toggle in dashboard settings to simulate posts without publishing.
  </Expandable>
</ExpandableGroup>
