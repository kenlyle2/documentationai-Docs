<Update label="2026-04-15" tags={["feature", "vault"]}>
## Media Vault archive/unarchive

Images can now be archived from the hover overlay or the detail dialog. Archived images appear under the **Archived** filter and are excluded from default Vault Cleaner operations. Unarchive restores them to the **All** view instantly.
</Update>

<Update label="2026-04-15" tags={["feature", "search"]}>
## Full-text image search

Vault search now uses Postgres full-text search over AI tags and descriptions. Supports phrase matching (`"red dragon"`), OR queries (`dragon OR eagle`), and exclusions (`dragon -eagle`). Best-quality images surface first, ordered by quality score.
</Update>

<Update label="2026-04-14" tags={["feature", "vault", "wordpress"]}>
## SearchIQ & WordPress gallery sync

Every AI-tagged image now syncs to the client's WordPress subsite as a stub post. SearchIQ indexes these stubs, making vault images semantically searchable inside WordPress. Gallery tokens secure the sync endpoint with ~192 bits of entropy.
</Update>

<Update label="2026-04-13" tags={["improvement", "credits"]}>
## Atomic credit deduction

Credits are now deducted via a Postgres function using `SELECT ... FOR UPDATE` to prevent race conditions. Two simultaneous generation requests can no longer both succeed when only one credit remains.
</Update>

<Update label="2026-04-12" tags={["feature", "auth"]}>
## Anonymous session onboarding

The `/start` onboarding flow now uses anonymous sessions. Users complete enrichment, GMB confirmation, profile creation, and first post generation before creating an account. At claim, all data is preserved under the same user ID — no re-entry required.
</Update>

<Update label="2026-04-10" tags={["feature", "wordpress"]}>
## Billing bridge

PostGlider users can now access their billing page with a single click. The session bridge authenticates them transparently, eliminating any separate WordPress login screen entirely.
</Update>

<Update label="2026-04-08" tags={["feature", "intelligence"]}>
## Competitor intelligence

Competitor social signals are now collected and surfaced as just-in-time nudges on the Generate page — prompting content creation when competitors post during a gap.
</Update>

<Update label="2026-04-05" tags={["feature", "ai"]}>
## Quality scoring

A dedicated AI call now scores each vault image on marketing quality (1–10 scale + PASS/REJECT). WordPress gallery search orders results by quality score so the best images surface first. Batch scanning available via the Tools page.
</Update>

<Update label="2026-04-01" tags={["feature", "vault"]}>
## GMB photo import

Google Business Profile photos are now imported directly into the Media Vault during onboarding and on-demand from the vault header. Photos are AI-tagged automatically on import.
</Update>
